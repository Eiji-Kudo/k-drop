# v2 Icon Guidelines

`app/v2` の UI アイコンは、情報アイコンとして絵文字を使わず、`lucide-react` ベースで統一する。

## ルール

- ホームカード、タブ、ページヘッダー、空状態では絵文字を使わない
- 基本サイズは `16px`, `20px`, `24px` の 3 段階に寄せる
- stroke 幅は `2.0` 前後で揃える
- アイコン単体を置かず、必要に応じて淡い面 `bg-surface-strong` や白半透明面に載せる
- アイコン色だけで状態差を作らず、面、枠線、ラベルも合わせて変える
- 装飾目的のイラストと、操作意味を持つ UI アイコンは分けて扱う

## この issue で反映した箇所

- ホームカードの絵文字撤廃
- ボトムタブのアイコン面と active 状態の統一
- プロフィール、クイズ開始、ホームヒーローの記号表現統一
