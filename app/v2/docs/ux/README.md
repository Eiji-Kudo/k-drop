# v2 UX Docs

`app/v2/docs/ux/` は、v2 フロントエンドの UI/UX とモチベーション設計に関するドキュメントをまとめるディレクトリ。

## 収録ドキュメント

- `ui-design-brushup-plan.md`
  - v2 全体の UI/UX ブラッシュアップ計画
- `gamification-plan.md`
  - 成長、競争、報酬ループの設計方針
- `motivation-ui-mvp-spec.md`
  - ホーム / クイズ結果の MVP 仕様
- `social-growth-ui-spec.md`
  - ランキング / プロフィールの競争・蓄積 UI 仕様

## 使い分け

- 全体方針を見るときは `ui-design-brushup-plan.md`
- モチベ設計を見るときは `gamification-plan.md`
- 画面単位の具体仕様を見るときは `motivation-ui-mvp-spec.md` と `social-growth-ui-spec.md`
